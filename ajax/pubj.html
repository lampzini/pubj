OVERVIEW

  • 웹퍼블리셔로서의 개인역량, 작업결과물을 전시하고자, HTML5/CSS3 기반으로 작업한 개인 포트폴리오 웹사이트 입니다.
  • 웹표준 및 웹접근성을 염두한 하드코딩으로, 활용도 높은 오픈소스 스크립트를 분석하여, 이를 이용한 포트폴리오 사이트를 제작하고자 하였습니다.
  • html/css 작업규칙을 체계화 하고자 코딩 컨벤션을 참조하여 코드 가독성 증진 및 협업에 필요한 일관된 코드 스타일을 유지하는 코딩을 하였습니다.
  • 또한, 처음 사이트 디자인을 할 때, 12 column grid 기반의 레이아웃 구조를 바탕으로 하여, 좀 더 정돈된 디자인을 목표로 사이트 작업을 진행하였습니다.
  • 포트폴리오 목록의 경우 Vue.js를 통해 개념을 익히는 작업을 수행하였습니다.
  • 스타일 적용에 있어서, 추후 레이아웃의 수정 및 반응형으로의 전환이 용이할 수 있게 Symantic Tag & Class Naming Control에 중점을 두고 CSS 스타일 작업을 하였습니다.

OVERVIEW

  • 관공서 사이트 특유의 다양한 페이지 구성에 대한 제작 감각을 익히기 위한 웹사이트 입니다.
  • html/css 작업규칙을 이끌어내기 위한 웹퍼블리싱 가이드라인 구조를 연습한 사이트 입니다.(html - 표준문법사용, css - id/class 네이밍규칙, 규칙적 스타일적용 등)
  • 서브 페이지 구성에 있어서 템플릿화 하여, 좀 더 수월한 디자인 및 코딩작업을 진행 하였습니다.
  • 관공서 사이트의 주요 지침인 웹표준, 웹접근성을 기반으로 코딩하였습니다.
  • javascript/jQuery를 사용하여, 공통 항목들(게시물)을 장르별로 축약하고 레이아웃을 정돈하였으며, 이를 통해 사용자가 주요 접근 항목으로의 빠른 이동이 가능하도록 하였습니다.
